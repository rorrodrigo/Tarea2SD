Actividad 1:
  Al seleccionar la opcion del menu principal se imprimen los mensajes recibidos cuando el usuario no está en la sala de chat, si se   
  quiere refrescar los mensajes recibidos se debe enviar un mensaje con el formato correcpondiente o presionar enter.
  No cumplir el formato de receptor@mensaje desencadenará la detencion de la consola.
  Cualquier error generado por no cumplir alguna instruccion detendrá la ejecucion, haciendo que el username(id) quede tomado en el      
  servidor y no se pueda volver a acceder con ese id a menos de que se reinicie el servidor.
